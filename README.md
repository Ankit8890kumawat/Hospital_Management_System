 #Hospital Management System

A Hospital Management System built to manage hospital operations such as departments, doctors, patients, and appointments efficiently.  
This project is designed to simplify hospital workflows and improve data management using modern backend technologies.


#Features

- Manage Departments
- Manage Doctors
- Manage Patients
- Appointment Scheduling
- RESTful APIs
- Database Integration
- Scalable Backend Architecture

- Technologies Used

- Java
- Spring Boot
- Spring Data JPA
- Hibernate
- REST API
- MySQL / PostgreSQL
- Maven
- Lombok
